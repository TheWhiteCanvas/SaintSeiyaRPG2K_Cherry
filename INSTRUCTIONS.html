Instructions:

[EN-US]:

Move the Game.exe file to your project folder and run it
If you need to change the settings, edit the file RPG_RT.ini

Insert this line to start the game in stretched mode:
Stretched=1

Insert this line to allow the player to toggle between aspect ratio and stretched with the F3 key:
F3Enabled=1

Every time you press the key the configuration will be automatically saved in RPG_RT.ini,
so the player will not need to press the key again the next time he opens the game.

Change Screenshot key, the options are F1, F2 (Default), F6, F7, F8, F9, F10, F11, 0 to disable
Screenshotkey=F1 → Example
Screenshotkey=0 → Disabled
(Any other value will be considered ZERO)


Change ScreenshotFolder path, examples:
ScreenshotPath=%UserDocs%
ScreenshotPath=%UserPics%

All options:
%UserDocs%            → C:\users\{UserFolder}\Documents\{GameDirName}
%UserPics%            → C:\users\{UserFolder}\Pictures\Screenshots\{GameDirName}
%UserRoaming%         → C:\Users\{UserFolder}\AppData\Roaming\{GameDirName}

Change ScreenshotFolder name, examples:
(Does not work with variables or symbols)
ScreenshotFolder=Snapshot
ScreenshotFolder=_Screenshot
ScreenshotFolder=Photos
ScreenshotFolder=Capture
ScreenshotFolder=%WinDir% → C:\users\{UserFolder}\Documents\WinDir

Example in RPG_RT.ini:

[RPG_RT]
GameTitle=My Adventure
MapEditMode=2
MapEditZoom=0
Fullpackageflag=1
F3Enabled=1
Stretched=0
ScreenshotPath=%UserDocs%
ScreenshotFolder=Snapshot

Instruções:


[PT-BR]:

Mova o arquivo Game.exe para a pasta do seu projeto e execute-o
Se precisar alterar as configurações edite o arquivo RPG_RT.exe

Insira esta linha para iniciar o jogo no modo estendido:
Stretched=1

Insira esta linha para permitir que o player alterne entre a modo tela proporcional e esticada com a tecla F3:
F3Enabled=1

Toda vez que pressionar a tecla a configuração ela será salva automaticamente no RPG_RT.ini,
então o jogador não precisará apertar a tecla novamente na próxima vez que abrir o jogo.

Mudar a tecla de captura, as opções são F1, F2 (Default), F2, F6, F7, F8, F9, F10, F11 e 0 para desabilitar.
Screenshotkey=F1 → Exemplo
Screenshotkey=0 → Desabilita
(Qualquer outro valor será considerado ZERO)


Mudando o caminho para a pasta de capturas de tela, exemplos:
ScreenshotPath=%UserDocs%
ScreenshotPath=%UserPics%

Todas as opções:
%UserDocs%          → C:\users\{Pasta do usuário}\Documents\{Mesmo nome da pasta do jogo}
%UserPics%          → C:\users\{Pasta do usuário}\Pictures\Screenshots\{Mesmo nome da pasta do jogo}
%UserRoaming%       → C:\Users\{Pasta do usuário}\AppData\Roaming\{Mesmo nome da pasta do jogo}


Alterando o nome da pasta aonde serão salvas as capturas de tela:
(Não aceita variaveis ou simbolos)
ScreenshotFolder=Snapshot
ScreenshotFolder=_Screenshot
ScreenshotFolder=Fotos
ScreenshotFolder=Capturas
ScreenshotFolder=%WinDir% → C:\users\{Pasta do usuário}\Documents\WinDir

Exemplo no RPG_RT.ini:
[RPG_RT]
GameTitle=Minha Aventura
MapEditMode=2
MapEditZoom=0
Fullpackageflag=1
F3Enabled=1
Stretched=0
ScreenshotPath=%UserDocs%
ScreenshotFolder=Capturas
